## Virtual Museum
- **What is it about:** This is the second project i completed, and i created it to fulfill an assignment given by the Art professor about the Impressionism. Its structure rapresents what could be seen as 
a virtual museum, with several rooms that contain different art styles and artist. Note that the text is in Italian.
- **Developers:** The software and the model of the museum are entirely done by me. Other components of the group (of 3 people) mainly gathered information for me to put them in.
- **Features:** Different versions of the project used various types of *illumination*, but i decided to remove it because it could have been demanding for the computer it should have ran on.
It is entirely in 3D, and it uses collisions and physics for the player and the *terrain component* with basic enviroment.
- **What i learned:** The use of Light in 3d ambients, the Terrain component, basic *3d modeling* and basic enviromental design.
