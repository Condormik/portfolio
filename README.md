# Portfolio
## Portfolio of my past Unity projects
This repository contains the projects i worked on. These projects range from about five years ago to last year. There are also other work in progress projects.

## Virtual Museum
- **What is it about:** This is the second project i completed, and i created it to fulfill an assignment given by the Art professor about the Impressionism. Its structure represents what could be seen as 
a virtual museum, with several rooms that contain different art styles and artists. Note that the text is in Italian.
- **Developers:** The software and the model of the museum are entirely developed by me. Other components of the group (of 3 people) mainly gathered information for me to put them in.
- **Features:** Different versions of the project used various types of *illumination*, but i decided to remove it because it could have been demanding for the computer it should have ran on.
It is entirely in 3D, and it uses collisions and physics for the player and the *terrain component* with basic environment.
- **What i learned:** The use of Light in 3D environments, the Terrain component, basic *3d modeling* and basic enviromental design.
- **Video:** [video link](https://youtu.be/nd_e7rBOVFA).

## Planet Merge
- **What is it about:** Planet merge is a mobile game that took inspiration from games such as "Fruit merge" with the addition of the space related theme. 
- **Developers:** I am the only developer of this project.
- **Features:** Implemented collision system and merging logic between planets, along with physics-based interactions.
- **What i learned:** I learned how to use unity to develop mobile games and add a basic ui and music system to the game.
- **Video:** [video link](https://youtu.be/uCKa60-FwiQ) (the music is not in-game).

## The time Engineer
- **What is it about:** It is a serious game about the history of computer science, exploring some important steps from its very beginning to the Apollo 11. (More additions could come in the future)
- **Developers:** It was created on a game jam. We were a group of 2, where the other component of the team made all the art and i developed all the code and designed the levels.
- **Features:** Different minigames with various approaches (top down, isometric, point-and-click)
- **What i learned:** I learned how to work in a small group, cooperating with others to achieve a goal in a brief time.
- **Video:** [video link](https://youtu.be/iLWaelH7C1s).
